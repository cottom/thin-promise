# Promise polyfill

## API

+ new Promise
+ Promise.prototype.then(onFulfilled?, onRejected?)
+ Promise.prototype.catch(errorhandler)
+ Promise.prototype.finally(finallyFun)
+ Promise.all(Array)
+ Promise.resolve
+ Promise.reject

## import

```js
import Promise from 'promise'

```
or

```html
<script src="cdn-url"></script>
window.Promise = ThinPromise
``
